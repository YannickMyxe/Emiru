About this lesson

This lesson is about the Japanese Alphabet.
The Japanese Language, unklike the rest of most western languages, doesn't use the traditional A-Z characters instead you have 3 kind of writing systems (actualy 4(Romanji) but the 4th one is just the Western Alphabet):

  • Kanji
  • Katakana
  • Hiragana
Now nowing how the Japanes writing system works, we can now look in more detail to them, but first of all,"Wich writing system should i learn first?". This is a very common question and the awnser is pretty simple, it doesn't realy matter, but still there is a better way of learning the Japanese Language. First of all you should probably learn Hiragana since this is one of easyer ones. This is mosly uses for non Japanese origion words, like computers etc. This is also used to show people how to pronounce a Kanji (Most of the time this is used when a new Kanji is introduced). Altough the writing systems don't uses spaces, comma's, points and questionmarks, they sometimes are added to make it easyer to read (this is mostly done to not native Japanese speakers)

Kanji:
In kanji each character has their own meaning, unlike the western languages where we use letters to make a word, this means that "tree"(木) and "father"(父) have their own unike character. In the western languages we make up the words using letters instead of unique characters, this is also true for the other 2 Japanes writing systems. A single character in Kanji is simply called Kanji.
Examples:
  • 明, meaning clear - bright - light
  • 酒, meaning sake(Alcoholic drink)
  • 水, meaning water
