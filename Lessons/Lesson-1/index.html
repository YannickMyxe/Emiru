About this lesson

This lesson is about the Japanese Alphabet.
The Japanese Language, unklike the rest of most western languages, doesn't use the traditional A-Z characters instead you have 3 kind of writing systems:

  • Kanji
  • Katakana
  • Hiragana

Kanji:
In kanji each character has their own meaning, unlike the western languages where we use letters to make a word, this means that "tree"(木) and "father"(父) have their own unike character. In the western languages we make up the words using letters instead of unique characters, this is also true for the other 2 Japanes writing systems. A single character in Kanji is simply called Kanji.
Examples:
  • 明, meaning clear - bright - light
  • 酒, meaning sake(Alcoholic drink)
  • 水, meaning water
